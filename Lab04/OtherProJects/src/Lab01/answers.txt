Trả lời câu hỏi

6.1
	- Nếu người dùng chọn Cancel thì in ra : “You've chosen: No”
	- Chỉ hiển thị 2 lựa chọn "Yes" và "No" thì phải thêm tham số JOptionPane.YES_NO_OPTION:
		int option = JOptionPane.showConfirmDialog(
		    null, 
		    "Do you want to change to the first class ticket?", 
		    "Select an Option", 
		    JOptionPane.YES_NO_OPTION
		);
	- Nếu muốn tùy chỉnh chữ trên nút thì ta phải dùng một mảng để lưu các kí tự tùy chỉnh:
		Object[] options = {"I do", "I don't"};
			
			int customOption = JOptionPane.showOptionDialog(null,
				"Do you want to change to the first class ticket?",
				"Customized Dialog",
				JOptionPane.YES_NO_OPTION,
				JOptionPane.QUESTION_MESSAGE,
				null,     // Không dùng icon riêng
				options,  // Mảng các nút tùy chỉnh
				options[0]); // Nút mặc định

			// Vì options[0] là "I do", nên nếu customOption == 0 tức là chọn "I do"
			JOptionPane.showMessageDialog(null, "You've chosen: " 
				+ (customOption == 0 ? "I do" : "I don't"));
